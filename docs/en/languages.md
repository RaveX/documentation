[FireSale](http://www.getfiresale.org) is already available in a number of the languages supported by [PyroCMS](http://www.pyrocms.com), the list is growing all the time and you can see all of the languages below. However we'd love to be able to offer our modules to a wider audience and if you're interested in translating it for the community you can submit a pull request on our [GitHub](http://www.github.com/firesale/firesale) repository.

### Available Languages
* [English](https://github.com/firesale/FireSale/tree/master/firesale/language/english)
* [Brazilian](https://github.com/firesale/FireSale/tree/master/firesale/language/brazilian)
* [French](https://github.com/firesale/FireSale/tree/master/firesale/language/french)
* [Indonesian](https://github.com/firesale/FireSale/tree/master/firesale/language/indonesian)
* [Italian](https://github.com/firesale/FireSale/tree/master/firesale/language/italian)
* [Lithuanian](https://github.com/firesale/FireSale/tree/master/firesale/language/lithuanian)
